# Finance – Finance & Project Accounting (Blueprint)

**Kod:** `finance`  
**Status:** Draft  
**Wersja:** 0.1

## Business Purpose

Rachunkowość projektowa dla długich cykli ETO (miesiące lub lata). Kluczowe: WIP, percentage of completion, revenue recognition według milestone'ów, actual costing na projekt.

## Aktualny Stan

Bardzo podstawowy (Chart of Accounts + Journal Entries). Brak project costing, AR/AP, fakturowania, fixed assets.

## Krytyczne Wymagania

- Project WIP + % of completion
- Milestone billing integration z TaxLegal
- Actual vs Planned costing (materiały + robocizna + overhead)
- Multi-currency na poziomie projektu
- Integracja z KSeF przez TaxLegalPBC (zgodnie z ADR-004)

Finance jest jednym z modułów, które muszą być bardzo mocne, żeby system był w ogóle używalny biznesowo.
